# Instruções de Instalação e Desenvolvimento

O único requisito é ter **Node.js e npm** instalados — [instale usando o nvm](https://github.com/nvm-sh/nvm#installing-and-updating).

Siga estes passos:

## Passo 3: Instale as dependências necessárias

npm i

## Passo 4: Inicie o servidor de desenvolvimento com auto-reload e pré-visualização instantânea

npm run dev

## Tecnologias utilizadas no projeto

Este projeto foi construído com:

- **Vite**
- **TypeScript**
- **React**
- **shadcn-ui**
- **Tailwind CSS**
